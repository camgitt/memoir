## Short Description (50 words)
memoir gives your AI tools persistent memory. It runs as a local MCP server inside Claude Code, Cursor, Windsurf, Gemini, and 7 more tools. Your AI can search, save, and recall context across sessions automatically. Sync across machines with one command. E2E encrypted. Open source.

## Long Description (150 words)
Every new AI coding session starts from zero. You re-explain your project setup, your architecture decisions, your preferences — every single time. memoir fixes this.

memoir is a CLI + MCP server that gives your AI tools persistent memory. Install it, add it to your editor's MCP config, and your AI can automatically search, save, and recall context across sessions.

It works with 11 tools: Claude Code, Cursor, Windsurf, Gemini CLI, GitHub Copilot, OpenAI Codex, ChatGPT, Aider, Zed, Cline, and Continue.dev.

Push your memory to Git or the cloud and restore on any machine. Translate memory between tools (migrate --from chatgpt --to claude). Session snapshots capture what you were working on so your AI picks up mid-conversation.

Everything is E2E encrypted (AES-256-GCM) with automatic secret scanning. MIT licensed. Zero infrastructure — just npm install.
